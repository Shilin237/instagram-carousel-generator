# /generate

Generate a new Instagram carousel for a brand.

## Steps
1. Ask for brand name, handle, primary color, logo, font preference, and tone
2. Derive the 6-token color palette from the primary color
3. Select typography pairing based on tone
4. Generate 7 slides following the slide architecture
5. Output a self-contained HTML file with swipe interaction

## Usage
/generate [topic] — e.g., /generate "5 tips for better sleep"
