# /brand-audit

Review a generated carousel for brand consistency.

## Checklist
- [ ] Primary color used consistently across slides
- [ ] Font pairing matches the chosen tone
- [ ] Brand name and handle present on first and last slides
- [ ] Progress bar present on all slides except last
- [ ] Swipe arrow absent on last slide
- [ ] CTA is clear and actionable on last slide
- [ ] No body text overlapping progress bar

## Usage
/brand-audit [filename.html]
