# Carousel Generator Skill

## Purpose
Transform any topic into a fully designed, 7-slide Instagram carousel as a self-contained HTML file.

## Inputs
- Brand name
- Instagram handle
- Primary hex color
- Logo (SVG path, initial, or none)
- Font preference (editorial / modern / warm / technical / bold)
- Tone (professional / casual / playful / bold / minimal)
- Topic / content brief

## Outputs
- Single `.html` file with all 7 slides
- Inline CSS using the 6-token brand palette
- Swipe interaction between slides
