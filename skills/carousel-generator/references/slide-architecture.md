# Slide Architecture Reference

| Slide | Type     | Background      | Key Elements                          |
|-------|----------|-----------------|---------------------------------------|
| 1     | Hero     | LIGHT_BG        | Brand name, hook headline, handle     |
| 2     | Problem  | DARK_BG         | Pain point, emotional resonance       |
| 3     | Solution | Brand gradient  | Core promise, transition statement    |
| 4     | Features | LIGHT_BG        | 3 key benefits with icons             |
| 5     | Details  | DARK_BG         | Supporting data or deep-dive          |
| 6     | How-to   | LIGHT_BG        | Step-by-step or checklist             |
| 7     | CTA      | Brand gradient  | Save/follow/link CTA, handle          |

## Universal Elements (every slide)
- Progress bar — top, color: BRAND_PRIMARY
- Slide number — subtle, top-right
- Swipe arrow — bottom-right (omit on slide 7)
- Bottom padding — min 48px to clear progress bar area

## Aspect Ratio
- 4:5 portrait — 1080×1350px at export
